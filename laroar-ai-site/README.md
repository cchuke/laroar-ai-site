# Laroar AI site

## Run locally
```bash
npm install
npm run dev
```

## Deploy on Vercel
1. Upload this folder to GitHub.
2. In Vercel, click **Add New > Project** and import the repository.
3. Vercel should auto-detect **Vite** and deploy with default settings.
4. For CLI deploy instead, run:
```bash
npm i -g vercel
vercel --prod
```
